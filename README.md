:pushpin: **이번 프로젝트로 성장한 점**  
[Before]  
● 유튜브에 나온 코드를 이해하지 못하고 컨닝해서 타이핑 하는 수준  
[After]  
● 이동 로직이나 애니메이션 제어 등 유니티의 기초 코드를 이해하게 되었기 때문에 컨닝하지 않아도 스스로 기초적인 로직을 입력할 수 있게 됨.  
● 코루틴의 사용법을 알게 되었고 이를 통해 연출 등의 동작에서 응용함.  
● PlayerPref을 사용하여 일부 변동 사항을 저장하거나 불러오는 방법을 알게 됨.
